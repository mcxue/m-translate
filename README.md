# A mini translation tool in Node.js

## Install

```bash
yarn global add m-translate
```
or
```bash
npm i -g m-translate
```

You will have a command —— fy


## Usage

```bash
fy <word>
```


**English → Chinese**
```bash
fy international
# it will return '国际的'
```

**Chinese → English**
```bash
fy 快乐
# it will return 'happy'
```
